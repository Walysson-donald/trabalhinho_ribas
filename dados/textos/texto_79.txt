Os conceitos fundamentais abrangem linguagens de marcação, estilo e programação. As linguagens principais incluem HTML (HyperText Markup Language), XML (Extensible Markup Language), CSS (Cascading Style Sheets) e JavaScript.HTML (HyperText Markup Language):HTML é a espinha dorsal da web e é usado para estruturar o conteúdo de uma página web.Consiste em uma série de elementos (tags) que envolvem o conteúdo para definir sua estrutura e significado.Exemplo de código HTML:<!DOCTYPE html><html lang="en"><head>    <meta charset="UTF-8">    <meta name="viewport" content="width=device-width, initial-scale=1.0">    <title>Exemplo de HTML</title></head><body>    <h1>Olá, mundo!</h1>    <p>Este é um parágrafo de exemplo.</p></body></html>2. CSS (Cascading Style Sheets):CSS é usado para estilizar o conteúdo HTML, controlando a apresentação visual das páginas web.Permite definir cores, fontes, espaçamento, layout e outros aspectos visuais.Ajuda a separar o conteúdo da apresentação, promovendo uma melhor manutenção e consistência.Exemplo de código CSS:body {    font-family: Arial, sans-serif;    background-color: #f0f0f0;}h1 {    color: #333;}p {    font-size: 16px;    line-height: 1.5;}3.
