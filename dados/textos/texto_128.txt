Título: Uma breve introdução sobre Computação de Alta Performance

Conteúdo:Breve contextualizaçãoAlguns termos importantesComputação de Alta PerformanceSaiba mais[ Breve contextualização ]Por alguns meses, fiz um estágio no Laboratório de Bioinformática e Biologia Computacional e tive que rodar umas análises usando o cluster de computadores.E isso foi completamente novo para mim.Comecei a pesquisar sobre o assunto e encontrei esse material organizado pela Harvard Chan Bioinformatics Core que traz uma breve introdução sobre Computação de Alta Performance.[ Alguns termos importantes ]Quando compramos um computador pessoal, como um notebook, algumas características que pesquisamos são:CPUTambém chamada de processador, a CPU (Central Processing Unit) é o “cérebro” do computador, é onde é feita a computação em si, como operações lógicas, aritméticas, processamento de dados etc.Processador Intel Core i5CoreA CPU pode ter um ou mais núcleos (cores, no inglês). As CPUs que têm apenas um núcleo precisam gerenciar as tarefas a serem executadas, visto que será executada uma tarefa por vez:Quando possuem dois ou mais núcleos, são capazes de realizar computações distintas simultaneamente (hoje a maioria dos processadores são multi-core):Exemplos de CPUs com dois ou mais núcleos.HDD e SSDExistem dois tipos de drives que podem armazenar os dados no nosso computador: HDD (hard disk drive) e SSD (solid state drive).A principal diferença entre os dois está em como os dados são armazenados e acessados.No HDD, os dados são armazenados e lidos de uma maneira mecânica, usando magnetismo. Já o SSD utiliza memória flash, o mesmo tipo de memória dos pendrives e cartões de memória.Exemplo de um HDD (à esquerda) e SSD (à direita).MemóriaÉ uma pequena quantidade de armazenamento de informações temporárias (voláteis). É onde os dados são temporariamente armazenados enquanto a computação é feita. Comumente chamada de memória RAM (Random Access Memory).Agora que vimos um pouco sobre esses termos, somos capazes de entender melhor a configuração da nossa máquina.
