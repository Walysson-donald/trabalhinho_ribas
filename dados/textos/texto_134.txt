Título: Introdução a Deep-Learning: RNN, LSTM e GRU

Seguindo os estudos sobre o dataset 3W, vamos avaliar o mesmo com outros tipos de modelo.Redes Neurais Recorrentes (RNNs) são um tipo de rede neural artificial especialmente projetada para lidar com dados sequenciais, como séries temporais, texto e música. As RNNs funcionam processando sequências de dados passo a passo, aprendendo as relações entre os elementos da sequência e utilizando essa informação para realizar previsões ou gerar novos dados.LSTM (Long Short-Term Memory) e GRU (Gated Recurrent Unit) são tipos específicos de RNNs que foram desenvolvidos para superar algumas das limitations das RNNs tradicionais, como o problema do gradiente evanescente.LSTM: Possuem uma estrutura interna mais complexa que as RNNs tradicionais, com células de memória que permitem armazenar informações por longos períodos de tempo. São especialmente eficazes em lidar com sequências longas e complexas, como textos em linguagem natural.GRU: São semelhantes às LSTMs, mas possuem uma estrutura mais simples e eficiente. São mais rápidas de treinar e podem ser mais eficazes em lidar com sequências curtas e menos complexas.Para todos os modelos, foi utilizado a mesma arquitetura do MLP e as respectivas funções para a extração de características segue abaixo:Os resultados para o dataset 3W seguem abaixo:RNN:LSTMGRU:Com os resultados obtidos, não é possível concluir efetivamente sobre os modelos.
