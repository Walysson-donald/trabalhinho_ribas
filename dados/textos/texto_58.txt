Porém, uma máquina incontrolável é inútil para fins de solucionar nossos problemas cotidianos. Nesse sentido, é necessário uma forma de controlar essa máquina e fazemos isso por meio de linguagens de programação.Linguagem de programação pode ser entendida como um conjunto formal de instruções que pode ser usado para controlar uma máquina. De modo simples, uma linguagem determina o que o computador vai fazer em cada momento e um programa é a execução concreta dessas instruções dessa linguagem. Se uma máquina fosse uma pessoa e disséssemos para ele ir de um ponto para o outro, tendo um buraco no meio do caminho, ele, inevitavelmente cairia, pois, por mais que o computador seja extremamente rápido, ele é extremamente literal na interpretação das instruções. O computador não faz julgamento crítico, ele irá executar o que foi determinado para ser executado.Diante disso, classificar linguagens é difícil, pois cada linguagem tem suas peculiaridades e forma de ordenar o computador de fazer algo.
